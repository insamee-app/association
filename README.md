# Associations

**Associations** is a platform to connect associations.

## Tech Stack

**Client:** Nuxtjs, Nuxt/Components, TailwindCSS

## Installation

This project uses `yarn`.

```bash
    # install dependencies
    yarn

    # dev server
    yarn dev

    # lint files
    yarn lint
```
